Similarities and Differences:

    Syntax:

  1. Both use { and } as code blocks
  2. Both use ; to end statements
  3. have a Math library e.g. Math.pow
  4. have return statements
  5. Both follow same code structure for functions and everything else
  6. Javascript uses "console.log()" instead of "System.out.println()", there is also a function called alert() in Javascript that allows you to display messages to the user
  7. In Javascript, you can use the prompt() function to get user input instead of scanner in Java

    Data Types:

  1. Primitive data types in Javascript are String, number, boolean, undefined and null. Non Primitive data types in Javascript are Objects and Arrays. This is different from Java because in Java a String is a Non primitive data type. Javascript also does not have seperate identifiers for an integer or double, it only has number.
  2. Unlike Java, Javascript has no reserved words for these datat types, it only has "var" and "let". These reserved words can store any data type. Examples include "var a = 50;" and "var b = "hello"; "

    Logic Controls:

  1. Javascript and Java have identiical Logic Controls, except Javascript also has a "===" control, which checks if its equal value and equal type.

    Loops and Conditionals:

  1. Javascript and Java have identiical Loops and conditionals, except Javascript also has a "for in" loop which loops through the properties of an object, and a "for of" loop which loops through the values of an iterable object

    Data Structures - Arrays and ArrayLists:

  1. There are no Arraylists in Javascript
  2. Arrays in Javascript are like arrays in Java but with properties from the ArrayList class. They are declared and function the same as their Java counterparts, except you are able to manipulate items in the array (I.E. Add, remove and show) using built in methods such as .push(), .pop(), and .shift()

    Class and Method Construction:

  1. Javascript methods and classes are identified by the function() reserved word and can be declared inside of variables
  2. JavaScript does not have polymorphic method declaration

    Object Handling:

  1. Object handling is identical to Java
